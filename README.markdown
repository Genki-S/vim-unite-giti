# vim-unite-giti

*vim-unite-giti* is unite source for using git.

## Usage

### Install

    NeoBundle 'git://github.com/kmnk/vim-unite-giti.git'


### Sources

- giti/status
- giti/branch


### Kinds

#### *giti/status* actions

- add
- reset
- commit
- amend
- checkout
- diff
- diff\_cached


#### *giti/branch* actions

- run
- delete
