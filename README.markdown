# vim-unite-giti

## Usage

### Install

    NeoBundle 'git://github.com/kmnk/vim-unite-giti.git'


### Command

    Unite giti/status
    Unite giti/branch


#### giti/status

- add
- reset
- checkout
- commit
- amend
- diff
- diff_cached


#### giti/branch

- run
- delete
